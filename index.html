GrafItby MatthewCallis

A plugin for ExpressionEngine to display super small, inline sparkline graphs/charts from CSV style data

A plugin for ExpressionEngine to display super small, inline sparkline graphs/charts from CSV style data. Using PHP it will generate a PNG image, using JavaScript it will work within a canvas element.
